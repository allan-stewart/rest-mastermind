REST Mastermind

This server hosts a RESTful API which can be used to play a mastermind-style game. Visit the base API endpoint to get started.

Playing the game

In order to play the game, you must first register as a player, then start a new game. The server will generate a 4-digit numeric secret value (for example 0380) which the player will try to guess.

The server will respond to each guess, letting the player know if they were correct. If the guess was incorrect, feedback is given indicating how many digits were in the correct position and how many were misplaced.

For example, if the secret value was 1132 and the player guesses 1234, the result will indicate that:

  • the guess was incorrect
  • two characters were in the correct position
  • one characters were misplaced

Example API call

The HTTP request to register a player should look similar to the following:

POST /api/player HTTP/1.1
Host: localhost:3000
Content-Type: application/json
Content-Length: 20

{"name": "John Doe"}

Here is an example curl request to generate such a request:

curl -v -X POST -H 'Content-Type: application/json' -d '{"name": "John Doe"}' http://localhost:3000/api/player
