REST Mastermind

This server hosts a RESTful API which can be used to play a mastermind-style game. Visit the base API endpoint to get started.

Playing the game

In order to play the game, you must first register as a player, then start a new game. The server will generate a 6-digit numeric secret value (for example "038047") which the player will try to guess.

The server will respond to each guess, letting the player know if they were correct. If the guess was incorrect, feedback is given indicating how many digits were in the correct position and how many were misplaced.

For example, if the secret value was "112223" and the player guesses "123456", the result will indicate that:

  • the guess was incorrect
  • one character was in the correct position (the 1)
  • two characters were misplaced (the 2 and 3)
